"Instructions"

Enter your preferred grid size below in values of tens from 1-100, if you enter a value that is lower than 0 or higher than 100, sketchboard won't appear. Whatever size you specify will determine the number of squares in each row and column. For example: if you enter "16" it will create a 16x16 grid for you (16 rows and 16 columns) forming a perfect square grid to bear your creativity !
